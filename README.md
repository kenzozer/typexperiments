## 🔧 Basic Troubleshooting

If typexperiments does not open, try these steps:

1. Right-click the file and choose Run as administrator
2. Make sure the file finished downloading
3. Check that you extracted the ZIP folder
4. Restart Windows and try again
5. Move the app to a simple folder like Desktop
6. Turn off blocking from your antivirus only if it stops the app from opening

If the screen stays blank:

- Close the app
- Open it again
- Try another display size
- Make sure your graphics drivers are up to date

If the animation looks slow:

- Close other apps
- Reduce the amount of text
- Lower the window size
- Try again after a restart
